# Dice Poker 開発プロセス

## 概要
ダイスポーカーは、5つのダイスを使用して役を競う古典的なゲームです。本プロジェクトでは、ミニマルで上質な「生活に溶け込むツール」としてのデザインを重視し、静かで心地よいプレイ体験を提供します。

## デザイン思想
- **静謐さ**: 派手な演出を避け、ダイスの質感（影、素材感）と動き（摩擦感、重み）で「触感」を表現する。
- **一貫性**: `testsites` 全体のテーマ（白、黒、細い線）を継承。
- **機能美**: 役の表示や再選択の操作を直感的に、かつ控えめに配置。

## 技術スタック
- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Architecture**: クリーンな状態管理とコンポーネント分割。
